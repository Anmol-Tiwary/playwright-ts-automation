# Playwright TS Automation

A Playwright TypeScript automation project scaffold with page objects, fixtures, UI tests, and API tests.

## Project Structure

- `src/pages/` - Page object classes
- `src/fixtures/fixtures.ts` - Playwright fixture composition
- `src/helpers/file-helpers.ts` - Shared file utilities
- `src/data/` - Test data files
- `tests/ui/` - UI test suites
- `tests/api/` - API test suites
- `.github/workflows/playwright.yml` - CI workflow

## Commands

- `npm install`
- `npx playwright install`
- `npm test`
- `npm run test:ui`
- `npm run test:api`
